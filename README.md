# TaskCalender
-TaskCalender is a website where one can type in their monthly goals/tasks. One can kind of mark the tasks their daily achieved task. Each task is displayed in table with different color. The theme of this site is different from my all other websites. This is mainly peach colored with calm llok.


# Features 
- The colors are different shades of peach.
- No specific theme i followed on this website like all my previous ones which were having rabbits all over the website.
- Index.html is simply the landing page with flower background image.
- On clicking a button on top-right you will be relocated to insert.html where you will be asked to provide the tasks or goals for every day till a month
- On clicking save you will be automatically relocated to calendar.hrml where all you have to do is fill the boxes under the date of a month with the colors assigned to them.\
- I added icon for the tab too which is not much clear but great.
- All websites have that flower falling effect which i think is my signature feature to add. (I added aparkles to all.)
- And yes at the end of the month you can reset the tavle as well as all the tasks in insert.html by simply clicking the button Reset all Tasks.


# The Languages Used
- HTML
- CSS
- JavaScript
